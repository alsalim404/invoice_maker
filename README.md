# Invoice Maker

Веб-приложение для быстрого выставления счетов:

- реестр своих юридических лиц;
- реестр контрагентов;
- выбор поставщика и покупателя в счете;
- позиции счета с количеством, ценой и НДС;
- генерация PDF в браузере;
- локальное хранение данных в браузере;
- экспорт и импорт резервной копии JSON.

## Локальный запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Деплой

В репозитории есть `Dockerfile`, поэтому Coolify может собрать и запустить приложение как Docker-приложение. Приложение слушает порт `80` внутри контейнера.
